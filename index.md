# pllst.xyz

Short links for playlist promotion.

This site hosts redirect pages to playlists, providing short, memorable URLs for easy sharing and promotion.

## Hosting

Hosted on GitHub Pages. Open for contributions.

## Contributing

Pull requests for new redirect links are welcome and will be served on a best-effort basis.
